# mokimo-web

Source for the [Mokimo](https://www.mokimo.co.uk) marketing website — static HTML/CSS/JS, no build step, no dependencies.

## Structure

```
├── index.html          # Main site
├── concepts/
│   └── index.html      # Concepts page
└── privacy/
    └── index.html      # Privacy policy
```

## Development

Edit the HTML files directly. Changes take effect on browser refresh.

## Deployment

Hosted on GitHub Pages at [www.mokimo.co.uk](https://www.mokimo.co.uk).
